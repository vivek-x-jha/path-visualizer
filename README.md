# Path Visualizer

A simple, visual high school-first planner for students who are still exploring possible college and career paths.

## What it emphasizes

- Flexible planning instead of locking into one path too early
- 8th grade through 12th grade journey view
- California A-G course snapshot
- Roadmap, checklist, and possibility explorer

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
